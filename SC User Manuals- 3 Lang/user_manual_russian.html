1. Введение

📚 Что такое школьная автоматическая система расписания?

Школьная автоматическая система расписания — это мощное интеллектуальное веб-приложение, предназначенное для упрощения и автоматизации сложной задачи составления школьных расписаний. Она управляет назначениями учителей и временными слотами, автоматически выявляя и предотвращая конфликты. Система поддерживает три режима составления расписания для любого стиля работы:

✨ Основные функции

🤖 Автоматический (Автозаполнение)

Система заполняет всё расписание автоматически одним нажатием, опираясь на ограничения, доступность учителей и требования предметов.

✋ Ручной (с подсказками)

Нажмите на любой свободный слот и выберите из умного списка только подходящих предметов и учителей — система скрывает недоступные и конфликтующие варианты.

🔀 Полуавтоматический (гибридный)

Сначала вручную разместите уроки, которые должны быть в фиксированных слотах, затем запустите автозаполнение, чтобы в автоматическом режиме заполнить все остальные слоты — лучшее из двух миров.

🤖 Три режима составления расписания

Автоматический, ручной (с подсказками) и гибридный — выберите подход, подходящий для вашего рабочего процесса.

👨‍🏫 Управление учителями

Управление учителями с назначениями предметов, недельными часами и временными слотами доступности.

👥 Поддержка подгрупп

Разделение классов на подгруппы или уровни, где несколько учителей преподают одновременно разным группам.

🔍 Обнаружение конфликтов

Проверка конфликтов в реальном времени предотвращает двойное бронирование учителей и нарушения ограничений.

📊 Множественные представления

Просмотр расписаний по группе (классу) или по учителю для полной прозрачности.

🌍 Многоязычность

Полная поддержка английского, иврита (RTL) и русского языков с более чем 200 переведенными строками.

📄 Варианты экспорта

Экспорт расписаний в PDF, Excel или печать непосредственно из браузера.

⚙️ Гибкие ограничения

Настройка максимума последовательных уроков, максимума уроков в день и пользовательских правил.

🏫 Кабинеты и местоположения

Определяйте классы, лаборатории и залы. Система отслеживает занятость кабинетов в реальном времени и показывает статус свободен/занят при назначении уроков.

🔗 Объединённые группы

Объединяйте несколько классов в одну группу для совместного урока или разделяйте класс на подгруппы для практических занятий — всё управляется автоматически.

💻 Системные требования

  • Браузер: Современный веб-браузер (Chrome, Firefox, Safari, Edge)
  • Интернет-соединение: Требуется для онлайн-версии
  • Разрешение экрана: Рекомендуется 1024x768 или выше
  • JavaScript: Должен быть включен

3. Создание расписаний

📊 Обзор входных данных

Школьная автоматическая система расписания требует три основных типа данных для создания расписаний:

1️⃣ Информация об учителях

  • Имя: Полное имя учителя
  • Предметы: Список предметов, которые они преподают, с часами в неделю и назначениями групп
  • Недельные часы: Временные диапазоны, когда учитель свободен (формат: День:ЧЧ:ММ-ЧЧ:ММ)
  • Доступность: Альтернативный формат с использованием номеров уроков (формат: День:1-5)
Пример записи учителя: Имя: Джон Смит Предметы: - Математика (5 часов, 10-А класс) - Физика (3 часа, 10-А класс) Недельные часы: Monday:08:00-12:00; Tuesday:08:00-15:00; Wednesday:13:00-17:00

2️⃣ Группы (Классы)

  • Просто список названий классов/групп
  • Примеры: "10-А класс", "11-Б класс", "Группа продвинутой математики"

3️⃣ Предметы

  • Название: Название предмета (Математика, Английский и т.д.)
  • Группа: Какому классу/группе принадлежит этот предмет
  • Часов в неделю: Сколько уроков в неделю
⚠️ Примечание о приоритете: Если заполнены как "Недельные часы", так и "Доступность", система будет использовать Недельные часы и игнорировать Доступность. Недельные часы имеют приоритет!

🔨 Создание расписания

  1. Перейдите на вкладку "Расписание групп"
  2. Выберите группу из выпадающего меню
  3. Нажмите кнопку "Автозаполнение"
  4. Подтвердите действие (оно перестроит расписание)
  5. Подождите немного, пока работает алгоритм
  6. Расписание появится в сетке!
✅ Успех! Если все настроено правильно, вы должны увидеть полное расписание со всеми заполненными уроками, цветовой кодировкой по предметам.

👥 Продвинутая функция: Поддержка подгрупп и параллельных занятий

🎯 Разделение классов на уровни и подгруппы!

Система поддерживает разделение одного класса на несколько подгрупп или уровней, обучение которых ведется одновременно разными учителями.

📚 Распространенные случаи использования:

  • Уровни математики: Продвинутая, обычная и базовая группы
  • Уровни английского: Разделение по уровню владения языком
  • Языки: Носители языка против изучающих как второй язык
  • Элективы: Студенты выбирают разные предметы в одно и то же время

⚙️ Как это работает:

  1. Создайте нескольких учителей: Определите всех учителей, которые будут вести параллельные уроки
  2. Назначьте одинаковый предмет и группу: Несколько учителей назначаются на один предмет для одного класса
  3. Умное планирование: Система находит общие доступные временные слоты для всех учителей
  4. Одновременные уроки: Создает уроки, где все учителя преподают свои подгруппы одновременно
  5. Обнаружение конфликтов: Предупреждает при добавлении уроков, которые делят несколько учителей
✅ Преимущества:
  • Автоматическая координация: Находит временные слоты, где все учителя доступны
  • Без двойного бронирования: Система понимает, что это параллельные, а не конфликтующие уроки
  • Гибкость в группах: Поддерживает любое количество подгрупп (2, 3 или более)
  • Обучение на основе уровня: Обеспечивает дифференцированное обучение
Пример:
  • У класса 10А математика, 5 часов в неделю
  • Учитель Джон преподает продвинутой группе
  • Учитель Сара преподает обычной группе
  • Учитель Майк преподает базовой группе
  • Система планирует всех 3 учителей одновременно на эти 5 часов

Вы можете загружать данные вручную через веб-интерфейс или импортировать из Excel. Редактируйте существующие расписания, просматривайте различные представления и управляйте всеми аспектами данных.

12. Кабинеты и места проведения

🏫 Общий обзор

Функция Кабинеты позволяет определять физические аудитории в школе (классы, лаборатории, спортзал и т.д.) и назначать их отдельным урокам. Система отслеживает занятость кабинетов в режиме реального времени и предотвращает назначение двух классов в один кабинет одновременно.

➕ Добавление кабинетов

  1. Откройте вкладку Кабинеты в главном меню.
  2. Нажмите Добавить кабинет.
  3. Введите название кабинета (например: «Лаб. 101», «Спортзал», «Библиотека»).
  4. При желании добавьте описание (например: «Компьютерный класс, 30 мест»).
  5. Нажмите Сохранить.
💡 Совет: Для каждой группы можно задать кабинет по умолчанию. При редактировании урока система автоматически предвыберет этот кабинет, экономя время.

🔗 Назначение кабинета по умолчанию для группы

Во вкладке Группы у каждой группы есть необязательное поле Кабинет. Укажите там домашнюю аудиторию группы. Тот кабинет будет автоматически подставляться при редактировании или назначении урока для этой группы.

📝 Назначение кабинета уроку

  1. В виде Расписание групп нажмите на любой временной слот.
  2. Откроется диалог Редактировать выбранное.
  3. В выпадающем списке Кабинет отображены все определённые кабинеты:
    • Кабинеты с ✓ — свободны в это время.
    • Кабинеты с ✗ — заняты, отображены серым.
  4. Выберите нужный кабинет и нажмите Сохранить.
✅ Предотвращение конфликтов: кабинет помечается занятым сразу после сохранения урока. Если вы пытаетесь назначить занятый кабинет, он будет виден. Назначение всё равно возможно, но требует внимания.

🖨️ Кабинет в экспорте PDF

При экспорте расписания в PDF назначенный кабинет печатается внутри ячейки урока, чтобы учителя и ученики сразу видели место проведения урока.

✏️ Редактирование и удаление кабинетов

  • Выберите кабинет из списка и нажмите Редактировать выбранное для переименования или изменения описания.
  • Нажмите Удалить выбранное для удаления кабинета. Уроки, назначенные в этот кабинет, автоматически очистят поле кабинета.
⚠️ Важно: Удаление кабинета не удаляет уроки — оно лишь убирает привязку к кабинету. Уроки остаются в расписании.

13. Объединённые группы (расширенные возможности)

🔗 Что такое объединённая группа?

Объединённая группа — это виртуальная группа, объединяющая два или более обычных классов (подгрупп) для проведения совместного урока. Эта мощная возможность охватывает два распространённых сценария:

📚 Сценарий А — Объединение классов

Два или более класса слишком малы или изучают один предмет (напр., электив). Они занимаются вместе в одном кабинете у одного учителя в один слот.

✂️ Сценарий Б — Разделение класса на подгруппы

Один класс делится на подгруппы для практических занятий (лаб., физкультура и т.д.). Каждая подгруппа занимается в разное время, но все они тяготеют к одной объединённой группе.

➕ Создание объединённой группы

  1. Откройте вкладку Группы.
  2. Нажмите Добавить группу.
  3. Введите описательное название (напр.: «10А+10Б Английский», «Лаб. 10 класс»).
  4. Поставьте галочку Объединённая группа.
  5. В поле Подгруппы выберите названия классов, входящих в эту группу.
  6. Нажмите Сохранить.
💡 Пример: Группы «10А» и «10Б» объединяются в «10А+10Б — Английская литература». Один учитель преподаёт оба класса вместе. Система автоматически расставяет обе группы на один слот.

📋 Назначение предметов для объединённой группы

Предметы назначаются объединённой группе, а не отдельным подгруппам. Во вкладке Предметы при добавлении укажите в поле Группа название объединённой группы. Система зачтёт все подгруппы участниками.

⚙️ Как система работает с объединёнными группами

  • Автозаполнение помещает урок одновременно во все подгруппы объединённой группы.
  • Расписание учителя обновляется один раз — учитель помечается занятым для всех объединённых групп.
  • Если подгруппа уже занята в данном слоте другой объединённой группой, слот считается занятым и пропускается.
  • Правило сиблингов: если подгруппы одной объединённой группы находятся в одном слоте для одного учителя — это один объединённый урок, а не конфликт.

✏️ Ручное редактирование урока объединённой группы

Нажмите на любой слот в расписании для одной из подгрупп. Диалог Редактировать выбранное покажет только подходящие предметы для данного слота, включая предметы из объединённых групп. Сохранение сразу обновляет все связанные подгруппы.

✅ Результат: Объединённые группы исключают необходимость вручную синхронизацию расписаний нескольких классов — система делает это автоматически.

⚠️ Агрессивное назначение (принудительное)

В исключительных случаях можно использовать функцию Агрессивное назначение в нижней части диалога Редактировать выбранное:

  1. Нажмите на слот, чтобы открыть Редактировать выбранное.
  2. Выберите нужный предмет в разделе Агрессивное назначение.
  3. Дважды нажмите на затемнённую область, чтобы разблокировать.
  4. Назначение будет сохранено, даже если оно нарушает обычные правила доступности.
⚠️ С осторожностью: агрессивное назначение обходит проверку конфликтов. Такие назначения отмечены в расписании и должны быть проверены перед публикацией.

📌 Рекомендации по объединённым группам

  • Давайте объединённым группам описательные названия (напр.: «10А+10Б — Физика»).
  • Сохраняйте последовательность в названиях подгрупп в соответствии со списком активных групп.
  • Внимательно настраивайте доступность учителя — он должен быть свободен для всех подгрупп.
  • Используйте вид Расписание учителя, чтобы проверить, что учитель не перегружен.
  • После финализации экспортируйте полное расписание в PDF для распределения всем подгруппам.
